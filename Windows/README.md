# VerseFlow Windows

## Overview

The Windows version of VerseFlow mirrors the desktop feature set of the macOS build, while being packaged for Windows installation and testing.

It is designed for:

- Windows desktop local playback
- lyrics and now playing viewing
- library browsing
- metadata enrichment
- play history recap and desktop theming

## Features

- local desktop library playback
- now playing screen
- lyrics screen
- play history
- metadata search for artists, albums, and songs
- desktop themes
