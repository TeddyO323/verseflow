# VerseFlow

VerseFlow is a local-first music player project built across Android and desktop experiences.

It is centered around one core idea: a cinematic local music player with rich metadata, strong now playing design, and a polished real-time lyrics experience.

VerseFlow currently spans multiple platform-specific builds, each with its own interface details, packaging flow, and release notes.

## What VerseFlow Includes

- local music playback
- synced and plain lyrics support
- metadata enrichment for artists, albums, and songs
- play history and listening recap features
- theme-driven interfaces
- Android phone, Android car, macOS, and Windows builds
