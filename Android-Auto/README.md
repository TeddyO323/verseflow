# VerseFlow Android Auto / Car

## Overview

The Android Auto / car-focused version of VerseFlow is designed for in-car local playback with simplified navigation, car-friendly now playing, and a dedicated lyrics experience.

It is intended for:

- car-installed Android testing
- Android Auto experimentation
- car now playing and lyrics layouts
- safer, reduced-focus browsing flows

## Features

- car-focused now playing screen
- car-focused lyrics screen
- local music playback
- library browsing for songs, albums, artists, playlists, folders, and genres
- dark, artwork-reactive playback surfaces
