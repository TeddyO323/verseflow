# VerseFlow macOS

## Overview

The macOS version of VerseFlow is the desktop experience for local-library playback, play history analysis, lyrics, metadata enrichment, and immersive theming.

It is designed for:

- desktop local playback
- immersive now playing and lyrics views
- metadata search for artists, albums, and songs
- play history recap and listening analysis

## Features

- local desktop library playback
- immersive now playing screen
- lyrics screen
- play history with recap views
- album, artist, and song metadata search
- theme presets including immersive and monochrome modes
