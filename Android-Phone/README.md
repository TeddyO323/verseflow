# VerseFlow Android Phone

## Overview

The Android phone version of VerseFlow is the main mobile experience for local music playback, lyrics viewing, metadata editing, and themed browsing.

It is designed for:

- full phone playback
- dedicated now playing and lyrics screens
- local-library browsing
- metadata editing and lookup
- play history and favorite-artist views

## Features

- local music playback
- full now playing screen
- synced and plain lyrics
- artist, album, and song metadata search
- play history
- favorite artists
- multiple themes
